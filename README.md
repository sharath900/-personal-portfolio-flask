# Personal Portfolio Flask

A responsive personal portfolio website built using Flask, HTML, CSS, and Bootstrap to showcase my skills, projects, education, and contact information.

## Features

- Clean and responsive portfolio design
- Home page with personal introduction
- Skills section
- Projects section
- Education section
- Contact section
- Flask backend routing
- Static files and templates structure

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run Locally

```bash
pip install -r requirements.txt
python app.py
